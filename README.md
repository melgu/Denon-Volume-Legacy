# Denon Volume

Control your Denon or Marantz receiver from the Keyboard, the Menu Bar or the Touch Bar.

## Keyboard Shortcuts

| Keyboard | Command                    |
| -------- | -------------------------- |
| ⌘+→      | Increase volume by 1 step  |
| ⌘+←      | Decrease volume by 1 step  |
| ⌘+↑      | Increase volume by 3 steps |
| ⌘+↓      | Decrease volume by 3 steps |
